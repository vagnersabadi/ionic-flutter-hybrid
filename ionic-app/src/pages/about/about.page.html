About — Ionic 7 Sobre o Projeto Estrategia de Migracao Ionic → Flutter

Este projeto demonstra como migrar gradualmente de Ionic 7 para Flutter, mantendo o app funcionando durante toda a transicao.

Como funciona:

  1. O Ionic 7 (Capacitor) e o host principal
  2. O Flutter roda como Add-to-App module
  3. Pastas android/ e ios/ sao compartilhadas
  4. O plugin FlutterRouterPlugin faz a ponte de navegacao
